# BudgiBook — Feuille de route

## Positionnement
BudgiBook = le carnet d’élevage simple pour les éleveurs d’oiseaux.

## Problème résolu
BudgiBook aide à éviter les carnets papier, les notes perdues, les photos dispersées et les oublis sur les oiseaux, couples et nichées.

## V1 — Minimum vendable
### Pages
- Dashboard
- Oiseaux
- Couples
- Nichées
- Profil élevage

### Oiseaux
Chaque oiseau doit contenir :
- Nom
- Numéro de bague
- Sexe
- Mutation
- Date de naissance
- Statut (actif / vendu / donné / décédé)
- Notes
- Plus tard : photo

### Couples
Doit permettre :
- associer un mâle et une femelle
- ajouter une note simple

### Nichées
Doit permettre :
- associer une nichée à un couple
- ajouter une date
- ajouter un statut
- ajouter des notes

### Dashboard
Doit afficher :
- nombre d’oiseaux
- nombre de couples
- nombre de nichées

### Profil élevage
Doit contenir :
- nom d’élevage
- région
- contact
- description

## Ce qu’on ne met pas en V1
- réseau social
- messagerie
- marketplace
- génétique avancée
- IA
- application mobile native
- statistiques complexes

## Monétisation recommandée
### Gratuit
- version limitée

### Payant (~4,99€/mois)
- version complète
- plus tard : photos + sauvegarde cloud

## Ordre intelligent de construction
1. Solidifier Oiseaux
2. Lier Couples aux vrais oiseaux
3. Lier Nichées aux vrais couples
4. Nettoyer Dashboard
5. Ajouter plus tard photos + cloud

## Règle produit
Toujours garder BudgiBook simple, rapide et clair.
